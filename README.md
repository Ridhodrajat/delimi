# Delimi - Sistem Informasi Terintegrasi

## Deskripsi Proyek
Delimi adalah sebuah aplikasi berbasis web yang saya kembangkan sebagai pemenuhan proyek implementasi akhir semester 3. Proyek ini merupakan hasil integrasi kolaboratif yang menggabungkan penerapan konsep arsitektur dari mata kuliah **Basis Data Lanjutan** dan **Desain Web**. 

Fokus utama dari pengembangan yang saya lakukan adalah menjembatani antarmuka pengguna (UI/UX) yang dinamis dengan sistem relasi *database* di bagian *backend*. Sebagai bentuk implementasi basis data, sistem otentikasi (*login*) pada aplikasi ini dirancang tidak menggunakan data statis, melainkan melakukan validasi secara langsung (*real-time query*) terhadap kredensial yang tersimpan dan direkam di dalam *database* MySQL.

## Fitur Utama
* **Integrasi Database Dinamis:** Menggunakan PHP Native untuk manipulasi dan pengambilan data (CRUD).
* **Desain Responsif:** Antarmuka web yang menyesuaikan dengan berbagai ukuran layar perangkat.
* **Sistem Otentikasi:** Validasi *login* terintegrasi penuh dengan tabel pengguna di basis data.

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP (Native)
* **Database:** MySQL
* **Environment:** XAMPP / Apache Web Server
