# Delimi - Personal Journal & Life Tracker Application

## Deskripsi Proyek
Delimi adalah sebuah aplikasi berbasis web yang saya kembangkan sebagai pemenuhan proyek implementasi akhir semester 3. Proyek ini merupakan hasil integrasi kolaboratif yang menggabungkan penerapan konsep dari mata kuliah **Basis Data Lanjutan** dan **Desain Web**.

Aplikasi ini dirancang sebagai platform pencatatan personal (*life tracker*) yang berfokus pada pengelolaan data aktivitas harian pengguna. Dari sudut pandang teknis, pengembangan aplikasi ini menjembatani antarmuka pengguna (UI/UX) yang responsif dengan arsitektur basis data relasional di bagian *backend*. Sistem otentikasi (*login*) pada aplikasi ini terintegrasi langsung dengan tabel pengguna, memastikan keamanan dan validasi data dilakukan secara dinamis melalui *real-time query* ke database MySQL.

## Fitur Utama
* **Jurnal Harian:** Fitur manajemen catatan aktivitas dan refleksi diri harian berbasis teks secara terstruktur.
* **Catatan Keuangan:** Modul pencatatan arus kas personal (pemasukan dan pengeluaran) untuk memantau kondisi finansial.
* **Mood Harian (Mood Tracker):** Fitur pelacakan kondisi emosional harian untuk membantu pengguna memantau tren kesehatan mental/psikologis mereka dari waktu ke waktu.
* **Sistem Otentikasi Dinamis:** Pintu gerbang login pengguna yang tervalidasi penuh dengan data kredensial di basis data.

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP (Native)
* **Database:** MySQL
* **Environment:** XAMPP / Apache Web Server
